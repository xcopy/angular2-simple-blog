Setup
-----
- Clone project `git clone git@github.com:xcopy/angular2-simple-blog.git`
- Change directory `cd angular2-simple-blog`
- Install packages `npm install`
- Run tests `ng test` or `ng e2e` (optional)
- Run dev server `ng serve`
